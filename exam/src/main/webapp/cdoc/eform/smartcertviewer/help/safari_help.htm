Safari 페이지 설정

  • 현재 Safari의 경우 페이지 여백 조정 옵션이 적용되지 않습니다. 따라서, PDF 문서로 출력되도록 설정되어 있습니다.
  •  
  • 고객님께서는 현재 애플 사파리 브라우저를 사용하고 계십니다.
  • 사파리는 브라우저 자체에서 페이지 설정 옵션이 정상적으로 적용되지 않습니다.
  • 올바른 인쇄를 위해 PDF 파일이 생성되고, 이를 이용해서
    인쇄하시기 바랍니다.
    인쇄 메뉴를 선택하시면, 자동으로 인쇄할 수 있는 전자서명된 PDF 문서가 내려받기 됩니다. 내려받기된 PDF문서는 편집을 할 수 없도록 되어있습니다.
